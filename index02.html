Exercicio 02 explorer.

Nesse desafio você irá criar uma lista de **estudantes** e, cada estudante dentro dessa lista, deverá conter os seguintes dados:

  1. [] Nome
  2. []Nota da primeira prova.
  3. [] Nota da segunda prova.
  4. [] Depos de criada a lista:
    1. [] Crie uma **função** que irá calcular a média das notas de cada aluno.
    2. [] Supondo que a média, para esse concurso é **7**, verifique **se** cada aluno obteve sucesso ou não em entrar no concurso e mostre uma mensagem na tela.
